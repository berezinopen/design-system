# Case Studies: Award-Winning Websites (2023–2025)

> Detailed breakdowns of 12 top-tier award winners, consolidated from all 3 research reports with duplicates merged.

---

## 1. Top Tier 3D Studio (v3) — Site of the Year 2023

| | |
|---|---|
| **URL** | [redacted-url] Tier 3D Studio.co |
| **Industry** | Creative studio / digital production |
| **Award** | Site of the Year 2023, Developer Site of the Year 2023 |
| **Aesthetic** | Futuristic, minimal, real-time 3D as primary visual language |

**Typography**: Clean geometric sans-serif system with high contrast between large headlines and small supporting copy. Type is a core design element.

**Color**: Dark base with bright neon-like accents from 3D scenes. Gradients and emissive materials against near-black backgrounds.

**Animation**: Complex WebGL scenes, cursor-responsive particles and displacement effects (Three.js forum confirmed ripple effect), camera moves tied to scroll, micro-interaction-rich intro sequences.

**Layout**: Multi-section narrative portfolio around full-viewport WebGL canvas. Content columns capped at ~1000px with narrower text blocks at 380–560px. Asymmetric grid clusters for project thumbnails. Large whitespace margins.

**Tech Stack**: Angular, Three.js, TypeScript, a popular Jamstack host, PWA capabilities. Entire site "entirely" in Three.js/WebGL with custom postprocessing shaders.

**Why it won**: Combines cutting-edge 3D with restrained UI — real-time graphics as brand language without drowning usability.

## 2. Vibrant Beverage E-commerce Maté — E-commerce Site of the Year 2023

| | |
|---|---|
| **URL** | [redacted-url] |
| **Industry** | Beverage / DTC e-commerce |
| **Award** | E-commerce Site of the Year 2023 |
| **Aesthetic** | Vibrant, playful, "funtional" (fun + functional) |

**Typography**: Friendly, rounded sans (New Montreal 2020) for headlines and body. Large, high-contrast type with contemporary energy.

**Color**: Saturated lime/olive greens, mustard yellows, oranges, accent pinks. Each flavor has its own sub-palette. Deliberately subverts the typical earthy organic product aesthetic.

**Animation**: Scroll-synchronized horizontal galleries, bouncy hover states on product cards, animated text scrambling on nav hovers, Lottie animations throughout.

**Layout**: Horizontal product carousels responding to vertical scroll ("sliding shelves"). bold color blocking with illustrated characters.

**Tech Stack**: a leading e-commerce platform + managed e-commerce payments, Lottie animations, heavily customized front-end.

**Why it won**: Balances creativity and clarity — sells product effectively while delivering a distinctive animated brand universe.

## 3. Award-Winning Digital Agency — Agency of the Year 2023

| | |
|---|---|
| **URL** | [redacted-url] Digital Agency.ca |
| **Industry** | Digital agency / studio |
| **Award** | Agency of the Year 2023 |
| **Aesthetic** | Editorial-meets-brutalist: asymmetric layouts, large type, layered imagery |

**Typography**: Bold grotesk display faces for headlines, simpler sans for body. Type used as a design object with animated tracking, masking, clipping. Custom PP Award-Winning Digital Agency New font (variant of PP Editorial New) — "font instead of logo."

**Color**: Generally dark or neutral bases with occasional bold color hits for projects and hovers.

**Animation**: Signature smooth scroll and parallax transitions wired into the layout. Carefully staged scene entrances/exits for project spotlights.

**Tech Stack**: Mix of a popular visual builder (client sites) + custom HTML/CSS/JS + GSAP + WebGL. Associated with open-source `Award-Winning Digital Agency-scroll` library.

**Why it won**: Consistent excellence across many immersive client projects; heavily influences scroll, navigation, and motion patterns in the award ecosystem.

## 4. Leading Storytelling Agency — Users' Choice Site of the Year 2023

| | |
|---|---|
| **URL** | [redacted-url] |
| **Industry** | Immersive web & storytelling design agency |
| **Award** | Site of the Day, Users' Choice Site of the Year 2023 |
| **Aesthetic** | Immersive, cinematic, 3D-forward with calm palettes |

**Typography**: Elegant sans-serif system with strong weight contrast. Token-driven, multi-product architecture. Large type anchors screens while 3D moves behind.

**Color**: Soft gradients and muted tones — lavenders, blues, neutrals. Occasional bright accents in 3D assets.

**Animation**: Smooth upward scrolling transitions from hero into projects. 3D scenes responding to scroll and cursor. Subtle parallax, micro-interactions on project cards. Later projects (Beat, ValenTime) adopt WebGPU.

**Layout**: Split hero (3D + text/CTA column). Bento-like case sections. Max-widths at 1024/1280px. 10–20px spacing increments.

**Tech Stack**: Three.js, WebGL/WebGPU, GSAP, Blender, Houdini. CSS shows 10, 20, 40, 60, 80, 140px spacing values.

**Why it won**: Blend of AI tech, subtle 3D, and simple, comprehensible navigation — inspiring model for storytelling agencies.

## 5. Premium Web3 Ecosystem Platform — Site of the Year 2024

| | |
|---|---|
| **URL** | [redacted-url] Web3 Ecosystem Platform.inc |
| **Industry** | Web3 / crypto ecosystem (prominent web3 IP) |
| **Award** | Site of the Year 2024, Developer Site of the Year 2024 |
| **Aesthetic** | High-end 3D: crystal structures, volumetric particles, bold type on dark |

**Typography**: Strong condensed display type + modern sans body. Shader-driven glitch and scramble effects on type while maintaining readability.

**Color**: Deep blues and blacks with icy cyan highlights. Glowing crystal enclosures as color focal points. UI chrome restrained.

**Animation**: Procedurally generated crystal growth, scroll-pinned 3D scenes, shader-driven UI text, VDB particle footer, slow-motion toggles, cursor interactions altering camera. Entire UI rendered via WebGL.

**Layout**: Continuous WebGL world structured as three-section scroll narrative. Text overlays positioned as layers on 3D. "Ice field" whitespace. Minimalistic labeling.

**Tech Stack**: Three.js, Svelte (some sources: Next.js + R3F), GSAP, Houdini, Blender. Custom VDB export tooling and shader iteration pipeline. Single WebGL canvas across all routes.

**Why it won**: Rare blend of fully immersive 3D and easy-to-navigate scroll interactions with first-class micro-interactions and performance-conscious engineering.

## 6. Pet Care Services Platform — Users' Choice Site of the Year 2024

| | |
|---|---|
| **URL** | https://www.premium award platforms.com/sites/dont-board-me |
| **Industry** | Pet care services |
| **Award** | Users' Choice Site of the Year 2024 |
| **Aesthetic** | Friendly, illustration-driven with "adorable" line art |

**Typography**: Rounded, approachable sans-serifs with clear hierarchy. Extensive mid-size headings and conversational copy.

**Color**: Soft neutrals + warm accents (muted yellows, oranges, greens). Light backgrounds for readability.

**Animation**: Micro-animations on icons and illustrations, simple section transitions. Intentionally avoids heavy distracting motion to keep booking flows straightforward.

**Layout**: Content-driven, practical booking flows with charming art direction.

**Tech Stack**: Modern JS framework with semantic HTML and CSS-based transitions (no heavy WebGL).

**Why it won**: Proves award-level doesn't require 3D — charming art direction, engaging copy, and practical UX that "never gets in the way" of the service.

| Pattern | Sites exemplifying it |
|---------|--------------------|
| **WebGL as primary medium** | Top Tier 3D Studio, Premium Web3 Ecosystem Platform, Leading Storytelling Agency (Labs)/Beat/ValenTime |
| **GSAP + ScrollTrigger core** | All 12 sites |
| **Dark-first aesthetic** | Top Tier 3D Studio, Premium Web3 Ecosystem Platform, Luxury Web3 Gaming Layer, AI Audio Experiential Microsite, Global Beverage Brand |
| **Editorial/content-drive** | Luxury Fashion Brand, Pet Care Services Platform, Award-Winning Digital Agency |

| **Playful without 3D** | Vibrant Beverage E-commerce Maté, Pet Care Services Platform |
| **Single-page narratives** | Premium Web3 Ecosystem Platform, Luxury Web3 Gaming Layer, AI Audio Experiential Microsite |
